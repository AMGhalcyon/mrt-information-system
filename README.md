# MRT Information System
A Python based MRT route plannning system that computes optimal route and displays it on a Turtle graphics window

# Features
- Shortest distance route
- Minimum travel time route
- Travel time estimate
- Fare calculation for different classes
- First and last train feasibility
- Interchange detection
- Route visualisation

# Algorithms 
- Graph-based modeling
- A* (Dijkstra variant using priority queue)
